# Student Management System - RIWI Python Test

A lightweight and efficient **Student Management System** built with Python. This project allows users to manage student records through a command-line interface (CLI), ensuring data integrity via custom input validations.

##  Features

- **Register Students**: Capture ID, Name, Age, Course, and Status.
- **List All Students**: View a formatted list of all enrolled students.
- **Search by ID**: Quickly find a specific student using their unique ID.
- **Update Records**: Modify existing student information.
- **Delete Students**: Remove student records from the system.
- **Data Validation**: Includes custom logic to ensure names and numbers are entered correctly.

##  Project Structure

- `main.py`: The core logic of the application and the menu system.
- `Fun_validations.py`: Contains validation functions (`validate_number`, `validate_text`) to prevent input errors.
